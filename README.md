# ⚙️ Generador Paramétrico de Rotores (DXF & Web)

Esta es una herramienta web interactiva diseñada para ingenieros mecánicos y diseñadores industriales. Permite generar el perfil de un eje segmentado a partir de datos técnicos, previsualizarlo en tiempo real y exportarlo directamente a un archivo **DXF** compatible con AutoCAD, SolidWorks y máquinas CNC.

## 🚀 Características principales
* **Interfaz Intuitiva:** Tabla editable tipo Excel para ingresar diámetros y longitudes.
* **Previsualización Dinámica:** El gráfico se actualiza instantáneamente con cada cambio.
* **Exportación a CAD:** Genera archivos DXF con capas y líneas de centro.
* **Colaborativo:** Genera enlaces únicos que contienen todo el diseño para compartir con colegas.

## 🛠️ Requisitos técnicos
Si deseas ejecutarlo de forma local, necesitarás:
* Python 3.8 o superior.
* Las librerías listadas en `requirements.txt`.

## 📦 Instalación Local
1.  Clona este repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/tu-repositorio.git](https://github.com/tu-usuario/tu-repositorio.git)
    ```
2.  Instala las dependencias:
    ```bash
    pip install -r requirements.txt
    ```
3.  Ejecuta la aplicación:
    ```bash
    streamlit run app.py
    ```

## 🌐 Despliegue en la Nube (Recomendado)
Para usar esta herramienta sin instalar nada:
1.  Sube este código a un repositorio de GitHub.
2.  Entra en [Streamlit Community Cloud](https://share.streamlit.io/).
3.  Conecta tu repositorio y despliega en un clic.
